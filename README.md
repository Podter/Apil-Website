# Apil-Website
Apple's Website

https://apil.podter.xyz

## Setup
Install Node.js (LTS is recommend)
```bash
# npm
npm i
# yarn
yarn
```

## Development
Run this and open `localhost:8080`
```bash
# npm
npm run dev
# yarn
yarn dev
```

# Start
Change port number in `package.json` and then run
```bash
# npm
npm start
# yarn
yarn start
```
